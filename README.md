## 仓库简介
GiHub图床，用于存储在线图片。
1. 文件结构：
    除了`README.md`文件外，按年份将图片创建目录。   
    >如2022年则创建目录`Pic2022`.

## 使用软件介绍
1. 软件名称：
    PicGo
2. 下载地址：
    https://github.com/Molunerfinn/PicGo
3. 软件设置：
    1. 找到软件，右键->打开详细窗口->图床设置->GitHub图床
    2. GitHub设置如下：
        + 设定仓库名： lukelee98/PicBed
        + 设定分支名：main
        + 设定Token：****************
        + 指定存储路径：Pic2022/
